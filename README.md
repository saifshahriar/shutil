# shutils: Shell Utilities

Inspired by [coreutils](https://www.gnu.org/software/coreutils/).

## Installation

Place the binaries in your path.

## Requirements

1. Unix-like systems preferred.
2. Deps:
   - POSIX Shell
   - `mawk`

## List of utilities

- `avg`
- `comma`
- `field`
- `freq`
- `total`
- `trim`
